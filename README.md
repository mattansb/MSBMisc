[here](https://mattansb.github.io/MSBMisc/reference/index.html).

## Install

You can install the package from [my
*R-Universe*](https://mattansb.r-universe.dev/):

``` r
install.packages('MSBMisc', repos = 'https://mattansb.r-universe.dev')
```
